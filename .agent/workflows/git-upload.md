---
description: 如何保存代码更改并上传到 GitHub
---

以后每次您写完代码，想要保存并上传时，请按照以下步骤操作：

### 1. 查看更改状态
查看哪些文件被修改了：
```bash
git status
```

### 2. 添加更改
将所有修改过的文件添加到暂存区：
```bash
git add .
```

### 3. 提交更改 (保存到本地)
编写一段简短的描述，说明这次改了什么：
```bash
git commit -m "简要描述您做了什么修改"
```
*例如： `git commit -m "修复了训练脚本的bug"`*

### 4. 推送到 GitHub (上传)
将本地的提交上传到远程服务器：
```bash
git push
```

---
**提示**: 如果您只是想保存到本地但不想上传，只需执行前三步即可。
